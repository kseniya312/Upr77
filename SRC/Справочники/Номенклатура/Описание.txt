В справочник номенклатуры заносится информация обо всех товарно-материальных
ценностях (ТМЦ) и услугах, включаемых в оформляемые в компании документы.
Справочник "Номенклатура" вызывается для ввода и редактирования информации из
пункта меню "Справочники" / "Номенклатура

В справочнике можно объединять позиции номенклатуры в группы и подгруппы.
Справочник реализован пятиуровневым. При заполнении справочника номенклатуры
используются дополнительные справочники-классификаторы, такие как "Единицы
измерения", "Виды деятельности", "Номера ГТД", "Страны". Все эти справочники
вызываются для ввода и редактирования информации из пункта меню "Справочники" /
"Классификаторы". 

Эти справочники можно заполнить предварительно или в процессе
ввода информации о номенклатуре и при оформлении документов.
Одна и та же позиция номенклатуры может иметь различные значения свойств, например,
в качестве значения свойства можно выбрать такие свойства, как "Импортные"
и "Отечественные". Одно из видов свойств можно назначить, как основное свойство.
Вид основного свойство назначается в пункте меню "Сервис" / Настройки параметров учета./
Закладка "Свойства". С помощью группировки по значениям основного свойства
можно проводить сравнительный анализ данных. 


Дополнительную информацию по позиции номенклатуры  можно получить с помощью 
соответствующих кнопок диалога:  

   По кнопке "Ценник ("Этикетка") можно распечатать ценник или этикетку для позиции 
номенклатуры, а по  кнопке "Прайс" распечатать прайс-лист. 
  
   По кнопке "Остатки" можно посмотреть информацию о текущих остатках конкретной позиции 
   номенклатуры  на складах, фирмах, в разрезе юридических лиц и упр. аналитики. 
   Можно посмотреть также остаток партии  купленных товаров и товаров, 
   принятых на реализацию (партии наличие). При выбранном варианте показа остатков 
   "партии, принятые на реализацию" показываются остатки партий комиссионных товаров, 
   которые были реализованы, но за которые еще не отчитались перед комитентом (не был 
   оформлен документ "Отчет комитенту"). При этом показываются те документы реализации, 
   по которым    был реализован комиссионный товар. При выбранном варианте показа остатков 
   "переданные на реализацию" показываются остатки товара у комиссионеров. 
 
 . По кнопке "Справочники"-"Свойства" можно просмотреть  и отредактировать список свойств, назначенных 
   данной позиции номенклатуры.  
 
  . По кнопке "Справочники"- "Аналоги" можно просмотреть и отредактировать  список аналогов, определенных 
    для данной позиции номенклатуры, во   внешних каталогах. 
  
  . По кнопке  "Справочники"-"Единицы" можно посмотреть и отредактировать список единиц измерения, 
    назначенных для позиции номенклатуры.
  
  . По кнопке  "Справочники"-"Цены" можно посмотреть и отредактировать  список цен, назначенных для 
    позиции номенклатуры. 
	
   . По кнопке  "Копировать" можно скопировать текущую позицию номенклатуры со всеми подчиненными 
     справочниками.   
	 
   . По кнопке "Поиск" можно вызвать обработку поиска элемента справочника по подстроке. 
      Поиск позиции номенклатуры  можно осуществить по краткому наименованию (наименование), 
      по коду или артикулу. Для осуществления поиска необходимо выбрать параметр, по 
      которому будет осуществляться поиск и ввести в подстроку для поиска текстовую информацию.
     Поиск осуществляется при нажатии на кнопку с изображением бинокля или при нажатии на кнопку 
     ENTER. Отобранные позиции будут отобраны в список найденных элементов. 
    Необходимо установить курсор на нужный элемент и нажать на кнопку "Выбрать". 
    Курсор установится на выбранный элемент в справочнике "Номенклатура". 
    История запросов сохраняется. При следующем поиске можно выбрать нужный 
    вариант запроса из списка, нажав на клавишу выбора.
   
ПОДБОР ПОЗИЦИЙ  В СПЕЦИФИКАЦИЮ ДОКУМЕНТА

При подборе позиций номенклатуры  в документ с помощью кнопки "Подбор" справочник 
"Номенклатура" открывается в режиме подбора. 

Окно подбора разделено на две части. В верхней части окна представлен список всех позиций 
номенклатуры в нижней части экрана будет отображаться список предварительно отобранных в 
документ позиций.
В  зависимости от установленного фильтра будут отображаться остатки позиций номенклатуры: 
по фирме, по конкретному складу, по юр. лицу,  по упр. аналитике или по компании в целом. 
С помощью кнопки "Изменить фильтр" можно изменить параметры установленного фильтра. 
Если в фильтре установлен флаг "Показывать только включенную в прайс номенклатуру", то в 
списке будут отображены только те позиции, для которых не установлен флаг 
"Не включать в прайс-лист". 

Подбор производится двойным щелчком мыши на нужной позиции, нажатием клавиши 
Enter или кнопки "выбрать" на панели инструментов справочника при выделенном 
в каталоге нужной позиции. 

В зависимости от выбранного варианта запроса количества, при подборе позиции 
будет дополнительно запрашиваться количество и цена позиции. 
Все выбранные товары записываются в список предварительно отобранных в документ позиций 
(список добавляемых позиций). 

С помощью кнопки "X" можно удалить конкретную позицию из списка. Если необходимо 
удалить все отобранные позиции в списке необходимо воспользоваться кнопкой "XX". 

В данном списке можно изменить количество, цену и сумму позиции, предварительно отобранной 
позиции для записи в документ. 

С помощью кнопки "Поиск" можно осуществить поиск нужной позиции номенклатуры. 
Поиск позиции номенклатуры  можно осуществить по краткому наименованию (наименование), 
по коду или артикулу. Для осуществления поиска необходимо выбрать параметр, по 
которому будет осуществляться поиск и ввести в подстроку для поиска текстовую информацию.
Список отобранных позиций номенклатуры появится в списке "Найденные элементы".  
Кнопка "Выбрать" или двойное нажатие мыши перенесет нужную позицию в список 
предварительно отобранных позиций.  
История запросов сохраняется. При следующем поиске можно выбрать нужный 
вариант запроса из списка, нажав на клавишу выбора.
                                  
Перенос товаров из предварительного списка в спецификацию документа осуществляется 
по кнопке "ОК".
