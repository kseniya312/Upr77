1. Описание вкладок.

При открытии обрабтки появляются пять вкладок.
 
1.2 Общий (таблицы). На данной вкладке отражены три таблицы заявками по каждому автомобилю отдельно.

1.3 Общий график. Отображается график, в строках которого находятся даты за выбранный период. В колонках отображается занятое  время. Крестом указывается занятое время,
 т.е. на данное время уже есть сформированные заявки по двум машинам компании.

1.4 Далее идут три вкладки по автомобилям отдельно. Информация аналогична вкладке "Общий график", только данные отражаются по каждому автомобилю отдельно.

1.5 Флажок "Заявки по автомобилю" - выводятся заявки только по автомобилю по текущей вкладке.

2. Заполнение заявок.

2.1 Одновременно с получением необходимой информации по доставке создается новая заявка.

2.2 Запрещено откладывать заполнение заявок на конец рабочего дня.

2.3 Кнопка "Создать" открывает форму новой заявки, в которой обязательна к заполнению вся информация по вкладкам - время начала доставки и время окончания, автомобиль, точка отправки, точка доставки, объем груза, документы основания, необходимый комментарий.

2.4 Дата (время) начала транспортировки. Время указывается в диапазоне от 9 до 23. Минуты от 0 до 60. Далее идет поле, где указывается дополнительная информация по времени.

2.5 Дата (время) окончания транспортировки. Время указывается в диапазоне от 9 до 23. Минуты от 0 до 60.  Далее идет поле, где  указывается дополнительная информация по времени.

2.6 Автомобиль.  Во вкладке выбирается  соответствующий автомобиль. Если автомобиль неизвестен, то необходимо выбрать вкладку "Заказная".

2.7 Точка отправки.  Указывается информация о точки отправки.

2.8 Контрагент (отпр.) Заполняется из справочника контрагентов.

2.9 Точка доставки. Указывается информация о точки отправки.

2.10 Контрагент (дост.) Контрагент, к которому доставляется груз из справочника контрагентов. Заполняется автоматически из документа основания.

2.11 Объем груза. Заполняется информация о товаре в произвольной форме.

2.12 Документы основания. Нажимая кнопку "добавить", выбирается документ основания. Выбранный документ автоматически отображается в нижней таблице.  При нажатии кнопки "открыть", данный документ доступен к просмотру.

2.13 Комментарий. Заполняется в произвольной форме. Описывается необходимая информация по доставке. Данная вкладка открыта к общему редактированию. При совпадении маршрута доставки с уже созданной  заявкой, разрешается вписывать комментарий по своей отгрузке в уже созданный документ.

3. Разрешены к редактированию только свои заявки. По заявкам, созданными другими сотрудниками введен запрет к редактированию. Исключение составляет графа комментарий.             

4. Заявки на следующий день заносятся до 18:00 текущего дня.                     
	 
5.	Обязательно производить контрольный звонок начальнику отдела логистики по несогласованным заявкам перед уходом с работы.      

6.	Редактирование созданных заявок производится только ответственным сотрудником отдела логистики при согласовании маршрутного плана.

7. Согласование вновь созданных заявок производится ответственным сотрудником отдела логистики с периодичностью в 2 часа.

8. При наличи флага "согласован", заявка приобретает статус обработанной и согласованной к отгрузке. В общих таблицах появляется пиктограмма "+" согласована.

9. Согласованные отделом логистики заявки последующему редактированию не подлежат.
