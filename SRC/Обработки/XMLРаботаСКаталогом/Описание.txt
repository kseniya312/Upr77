Режим просмотра каталога доступен в различных режимах работы: 
в списке и в карточке каталога, 
при загрузке коммерческих предложений,  при загрузке документов. 
Экран просмотра каталога универсален для всех режимов. 
На закладке "Общие" отображается список товаров в каталоге.
 Экран разделен на две части: в левой части представлено 
дерево каталога. При нажатии на отметку около группы начинает
 разворачиваться дерево каталога. 
При наличии в группе/подгруппе товаров в правой части экрана
будет развернут список товаров (идентификатор и наименование). 
При просмотре возможно отключение иерархического списка 
(кнопка, аналогичная , используемой при отображении 
справочников 1С:предприятия), в этом случае в правой части
экрана  будет представлен весь список товаров в каталоге. 
 В этом списке могут быть представлены и цены для товара,
 хранящиеся в справочнике цен, подчиненному справочнику 
"Номенклатура" или в файле каталога. Режимом показа цен 
можно управлять с помощью соответствующих переключателей 
на закладке "Цены". 
В списках "Из справочника цен" и "Из пакетов предложений" 
необходимо отметить те типы цен, которые хотелось бы просмотреть.
Возможно два режима представления цен:выводить значения цен
в колонках таблицы или в информационной строке. 
Режим просмотра в колонках таблицы очень удобно 
применять для сравнения цен поставщика и текущих 
закупочных цен, однако при большом объеме отображаемых позиций
это замедляет работу. 
Ограничить число отображаемых позиций 
можно с помощью настроек, представленных на закладке "Фильтр".
 Здесь можно установить фильтр по типу цен. При выборе типа цен 
из справочника цен, в списке будут представлены только те товары, 
которые имеют ненулевое значение данного типа цены. 
Можно также показать в списке только идентифицированные
 позиции (то есть которые имеют аналоги в справочнике 
"Номенклатура" для данного каталога) или не идентифицированные 
(не имеющие аналогов). 
Первый режим отображения удобен, 
например, при сравнении цен поставщика в каталоге и
 закупочных цен, а второй режим удобно применять 
в том случае, когда загружается документ, 
чтобы просмотреть не идентифицированные позиции документа.
Возможности представленные на закладке "Опции" 
применяются при загрузке документов для 
автоматического добавления или идентификации новых товаров.
