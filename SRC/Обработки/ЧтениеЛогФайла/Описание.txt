Обработка предназначена для чтения log-файлов, созданных прокси сервером
Kerio WinRoute. 
При формировании отчёта выполняется группировка по IP адресам, с которых
пользователи пользовались  Internet, а затем группировка по объёму скачанных
данных.
При необходимости можно выйти на web-страничку, посещение которой анализируется. 
Для этого следует дважды щёлкнуть левой клавишей мыши на соответствующем адресе 
в поле отчёта. При этом будет запущен Internet Explorer с заданной страницей.

Обработка работает с файлами, указываемыми в верхней строке адреса (выбор по 
кнопке "...").
Часто можно использовать кнопку быстрого выбора файла по дате (кнопка справа от 
кнопки выбора файла).
